# IT-HELPDESK-MANAGEMENT-SYSTEM
Designed and developed a web-based IT helpdesk system to log, track, and manage support tickets for an organisation Implemented features including ticket submission, priority classification (High/Medium/Low), status tracking (Open/In Progress/Resolved), and real-time filtering by status and priority Built a live dashboard displaying key metrics.
Features
📝 Ticket Submission – Users can log IT support requests
⚡ Priority Classification – Assign priority levels (High, Medium, Low)
🔄 Status Tracking – Manage ticket lifecycle (Open, In Progress, Resolved)
🔍 Real-Time Filtering – Filter tickets by priority and status
📊 Live Dashboard – View key metrics and system performance
💾 LocalStorage Integration – Persistent data without a backend
🧠 Project Objectives
Simulate a real-world IT service management system
Improve efficiency in handling and tracking support requests
Provide visibility into support operations through dashboards
Demonstrate frontend development and business problem-solving skills
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
📌 How It Works
Users submit a support ticket with relevant details
Each ticket is assigned a priority level and status
Tickets are displayed in a structured table
Users can filter tickets based on status or priority
A dashboard provides insights into ticket distribution and performance
